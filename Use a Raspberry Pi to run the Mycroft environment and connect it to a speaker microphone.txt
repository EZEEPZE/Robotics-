Step 1: Install Raspberry Pi OS (Lite – Recommended)
Raspberry Pi 3 has limited RAM, so Lite OS works best.
1. Open Raspberry Pi Imager
2. Select:
o OS → Raspberry Pi OS Lite (64-bit)
o Storage → microSD card
3. Click 
o Enable SSH
o Configure Wi-Fi
4. Flash OS and boot Raspberry Pi 3
Login:
username: pdl
password: pdl

Step 2: Connect Speaker & Microphone
Plug in your USB audio device.
Check detection:
arecord -l
aplay -l
If devices appear → continue


Step 3: Configure Audio (Very Important)
Open mixer:
alsamixer
 Press F6 → select USB audio
 Increase:
o Mic
o Speaker
 Press M to unmute
Test microphone:
arecord test.wav
aplay test.wav
If you hear your voice → audio OK


Step 4: Update Raspberry Pi
sudo apt update && sudo apt upgrade -y
sudo reboot


Step 5: Install Mycroft Environment (OVOS –
Lightweight Mode)
Install required dependencies
sudo apt install -y git python3 python3-pip python3-venv portaudio19-dev

Install OpenVoiceOS Core (Optimized for Pi 3)
git clone https://github.com/OpenVoiceOS/ovos-core.git
cd ovos-core
python3 -m venv .venv
source .venv/bin/activate
pip install --upgrade pip
pip install ovos-core[extras]
Takes 15–20 minutes on Pi 3 (Slow on Pi3)


Step 6: Install Wake Word & TTS
pip install ovos-dinkum-listener ovos-tts-plugin-pico
Pico TTS is fast & lightweight (ideal for Pi 3).


Step 7: Configure Audio & Wake Word
Create config:
mkdir -p ~/.config/OpenVoiceOS

nano ~/.config/OpenVoiceOS/mycroft.conf
Paste:
{
"listener": {
"wake_word": "hey mycroft"
},
"tts": {
"module": "ovos-tts-plugin-pico"
},
"audio": {
"input": {
"device": "default"
},
"output": {
"device": "default"
}
}
}
Save → CTRL+X, Y, ENTER


Step 8: Start Mycroft Services
Activate environment:
source ~/ovos-core/.venv/bin/activate
Start voice assistant:
ovos-core
Say:
“Hey Mycroft, what is the time?”
You should hear a response


Step 9: Auto-Start on Boot (Optional but Recommended)
Create service:
sudo nano /etc/systemd/system/ovos.service
Paste:
[Unit]
Description=OpenVoiceOS (Mycroft)
After=network.target sound.target
[Service]
User=pi
ExecStart=/home/pi/ovos-core/.venv/bin/ovos-core

WorkingDirectory=/home/pi/ovos-core
Restart=always
[Install]
WantedBy=multi-user.target
Enable:
sudo systemctl daemon-reload
sudo systemctl enable ovos
sudo systemctl start ovos
